# scoop-w42

Scoop bucket for the [w42 CLI](https://github.com/yarn-rp/web42-network).

## Install

```powershell
scoop bucket add w42 https://github.com/yarn-rp/scoop-w42
scoop install w42
```

## Update

```powershell
scoop update w42
```

This bucket is automatically updated by CI when a new CLI version is released.
